# Communauté du Marquis 12 — Site Web

## Structure du projet

```
projet/
├── index.html          ← Page principale
├── css/
│   └── style.css       ← Styles cyberpunk noir/rouge/bleu
├── js/
│   ├── background.js   ← Canvas animé (grille + ville + particules)
│   ├── main.js         ← Navigation, scroll, tabs
│   └── admin.js        ← Panneau admin (localStorage)
└── image/              ← Mets tes PNG ici !
    ├── icon_TacticalBoy.png
    ├── twitch.png
    ├── youtube.png
    ├── tiktok.png
    ├── instagram.png
    └── discord.png
```

## Accès au panneau Admin

- **Bouton** : Clique sur "ADMIN 🔒" en haut à droite
- **Raccourci** : `Ctrl + A`
- **Login** : `admin` / `1234`

## Déploiement GitHub Pages

1. Upload tout le dossier dans ton repo GitHub
2. Va dans Settings → Pages → Source: main branch
3. Le site sera sur `https://lemarquis12.github.io`

> ⚠️ Pense à remplacer `https://discord.gg/tonlien` par ton vrai lien Discord dans index.html
